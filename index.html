TaxiSecu - Help
  • 1. Initial Information
    • Download TaxiSecu App

    • Initially there are some Introductory Screens from where the App can be continued further for next eligible default page. After continuing Introductory Screens, There will a Message,
      “Read TaxiSecu user Help in Settings, Before Using the App.” The Message will be shown only first time before the user’s registration.
  • 2. User Registration notepad
    • Select Country – User has to select his correct country from the system’s in-built country list which can be opened on the click the event. User has to select right country because the system reckons the country ISD code to use in generating system SMS.
      User can Register himself only with the same Mobile Number where the App is to be downloaded.
      On entering Mobile Number and after click to continue, System prompt for a question that “Is this your own Mobile Number?” User has to press Yes or NO, If user press NO the system takes back to re-enter Mobile Number otherwise on pressing Yes, system continues for auto verifying process.
      System sends a SMS with a security Code on the mentioned mobile number which is subsequently read by the App automatically to verify the same mobile number. Thereafter a message is generated that the number is successfully verified by the system.

      In case of failure of mobile verification, App gives a message “ Sorry, the mentioned number doesn’t belong to the device.” Hence user can’t register himself from another’s mobile number.
  • 3. Manage Profile
    • User has to enter following mandatory values in the concerned fields
      - Enter First Name
      - Enter Last Name
      - Enter Email
       On click to continue, there comes a Message “The Profile Successfully Saved.” OK
  • 4. Contact Book, Favorites & Friends
    • From Phone Contact Lists Favorites List can be managed to expend / add up. The Friends list can be managed from Favorites to send concerned SMS especially on Start and Finish Riding etc.
      System picks the concerned Mobile Contacts after getting consent of the User. The App asks question “TaxiSecu need to access your address book to sync your Contact List.“ - Yes / No. If user permits then system pick the contacts in its own system for further usage to manage favorites in faster manner. Otherwise if user selects No then system reads the Contacts on RUN time to manage favorites.
  • 5. Manage Favorites – from Contact List
    • User can select and deselect the contact list for managing favorites. On selecting any contact, it is added in favorites’ lists. On deselecting any contact the system removes the same from favorites list. System also highlights the contacts which are already registered in TaxiSecu App.
  • 6. Manage Friends – from Favorites List
    • There are 2 categories of Favorites. First category is from the defaults favorites as those exiting TaxiSecu user(s) which have declared you as their friend. They will be your defaults favorites. User can’t remove them as they are managed by system. Second category of Favorites can be managed from contact lists. But the favorites which are added up from users mobile contacts can be removed on deselecting from manage favorites’ page in contacts list.
      User can select and deselect the Favorites list for managing friends for informing about Start and Finish Ride. On selecting any Favorite, it is included in friend(s) lists. On deselecting any friend the system removes the same from friend(s) list. There is a system defined limit on making friends from favorites
      However No Ride is Possible without having any friend(s). If you don’t have favorites then system by defaults, puts you at Manage Favorites Page to manage favorites. If you don’t have friends but have favorites then system by default puts you at Manage Friends Page.
  • 7. Start Ride
    • The pre-requisites of Start Ride are, User Registration, Completion of Profile and Final Selected Friends from Favorites List as mentioned previously in manage favorites and manage friends.
      User can start ride if user has friend(s). On the Start Ride Page Loading, the system attempts to pick the start GPS locations (Latitude and Longitude) in order to draw Google Map from Start Point to Finish Point of the last Ride.
      CAUTION: Firstly, System takes time in picking up the exact start GPS locations (Latitude and Longitude). Secondly, system can pick the start GPS locations (Latitude and Longitude) at open area or from vehicle. But the same can’t be traced easily under any roof or closed area.
      User has to enter Taxi Number. Once the GPS mentioned start information is traced by the Apps, user has to click Start Ride which followed by Finish Ride Screen.
      On Click to Start Ride, the SMS are/is being sent to the selected friend(s) from the favorites. Once the Ride Starts after completing all back end tasks, a message displayed as “The Ride Start Successfully!”
  • 8. Finish Ride
    • On Finish Ride Page System displays the running time of travelling right from the start of the ride to show the total ride time as of now.
      There will be an option to View SMS Alert Confirmation List to who the Start SMS had been sent on Start of the Ride. User has tap on the message to see the concerned list.
      On the Finish of the Ride User has to click on the click to finish the Ride. The Red color is used to finish the Ride to make the user more cautious to finish the ride. On clicking the Finish, there comes a message that Ride Finished Successfully. Thereafter Rating Page for as feedback about the Ride. After rating, user has to click to continue. On the click, a message comes as for completing the Ride successfully, enjoy your next Ride.
      Now the default next page is Start Ride as a iterative cycle to start and finish Rides.
